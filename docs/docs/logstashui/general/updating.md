# Updating

LogstashUI will notify you when a new version is available via a banner in the navigation sidebar.

To update LogstashUI to the latest version:

## Linux

```bash
cd logstashui/bin
./start_logstashui.sh --update
```

## Windows

```cmd
cd LogstashUI\bin
start_logstashui.bat --update
```
